Fluency 0.0.8

Thank you for beta testing Fluency!

Improvements and Fixes

  • Fluency remembers the last page you published a draft under and pre-selects that page
  • The time when a page draft was last edited is now immediately displayed when selecting the draft
  • Press shift-return to copy the link to a downloaded page in page search
  • You should no longer get an error about Apple being unable to check for malicious code when opening Fluency
  • Fix an issue where some parts of the user interface would fail to update after logging in from an expired session
  • Fix an issue where editing the page draft text would appear slow in some cases
  • Fix an issue where flickering would occur when editing a page draft
  • Fix an issue where opening web links would not work correctly in some cases
  • Fix an issue where the wrong quick-nav search result would sometimes be highlighted after clicking
  • Fix an issue where typing a space in the quick-nav bar would crash the app
