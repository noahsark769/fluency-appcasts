Fluency 0.1.0

Thank you for beta testing Fluency!

Improvements and Fixes

  • Fluency app menus now work correctly and dynamically update to reflect the page you're viewing
  • Fluency's default view now includes helpful links
  • Fluency now loads large spaces up to 1000% faster
  • Fixed an issue where images would sometimes render very small
  • Fixed an issue where user tags in pages would sometimes render cut-off until the second page load
  • Fixed an issue where Fluency would stay open when pressing cmd-Q at the login screen
  • Fixed an issue where if there was an error loading a page, Fluency would display as if the page was empty
  • Fixed an issue where links to blog posts would not display correctly in pages
  • Fixed an issue where login flow text fields were hard to see in light mode
