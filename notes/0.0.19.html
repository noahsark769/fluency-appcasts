Fluency 0.0.19

Thank you for beta testing Fluency!

Improvements and Fixes

  • Tables in pages now have better visual style and render correctly in almost all cases
  • Code blocks in pages now display a button to copy the content to the clipboard
  • Fluency now allows registering a global shortcut to open the new page window
  • Fluency now detects when you paste a link in the Markdown editor and selects the right characters so you can easily give the link a title without clicking back and forth
  • Fixed an issue where pasting into the page compose view would sometimes insert extra spaces
  • Fixed an issue where headings with status macros would not appear correctly in the tables of contents
  • Fixed an issue where an incorrect link would be copied for a heading if the heading contained a status macro
  • Fixed an issue where the button to favorite a page was invisible in light mode
