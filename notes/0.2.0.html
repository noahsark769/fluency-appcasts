Fluency 0.2.0

Thank you for beta testing Fluency!

Improvements and Fixes

  • Fluency now has menu bar items for reporting a bug, and bug reports are standardized to Fluency's support email
  • Fluency now allows multiple browser windows (create a new window with Window > New)
  • Fluency now has a menu item to fully quit the app, in addition to exiting to the menu bar
  • Fixed an issue where most Fluency windows could not be minimized
  • Fixed an issue where page deletion would not work on macOS Big Sur
  • Fixed an issue where white corners would appear around status macros in dark mode
  • Fixes an issue where status macros inside some tables would render incorrectly
  • Fixed an issue where copy, paste and some other editing commands would not work in the page compose screen
  • Fixed an issue where text fields would sometimes appear with the wrong font on macOS Big Sur
