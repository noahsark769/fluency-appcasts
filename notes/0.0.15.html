Fluency 0.0.15

Thank you for beta testing Fluency!

Improvements and Fixes

  • Colors for the page content and the page heading are now more cohesive
  • Fluency autocorrects colors so that you can view page content in Dark Mode if enabled on your system
  • Fluency tracks your recent pages for quick access in the sidebar
  • Fixed an issue where links to page attachments would not display
  • Fixed an issue where nested list items would not show up or would be incorrectly formatted
  • Fixed an issue where code blocks with embedded colors would have line breaks incorrectly formatted
  • Fixed an issue where code blocks and monospaced content would sometimes not display line breaks correctly
  • Fixed an issue where empty entries in the table of contents would sometimes appear
  • Fixed an issue where links to headings on the same page would sometimes not display
  • Fixed an issue where text inside of tip and callout macros would sometimes show with extra line breaks
  • Fixed an issue where nested bullet lists would sometimes not display correctly
  • Fixed an issue where periodic space updates would sometimes not start
