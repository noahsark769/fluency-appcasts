Fluency 0.0.18

Thank you for beta testing Fluency!

Improvements and Fixes

  • Fluency now allows copying a link to a heading through the context menu in the table of contents
  • Refreshing space pages is now available from the sidebar context menu
  • Fluency now allows deleting pages from the page tree view
  • Fluency now detects when you paste a url into a page draft and converts it into a link
  • Fluency settings have moved to the standard macOS app Preferences window
  • Fluency now allows setting preferences for page draft color styles
  • Fluency now resizes headings when creating a draft page for more visual differentiation
  • Fluency now uses a new font for code blocks which should make them easier to read
  • Fixes an issue where only the first 25 child pages of a page would be downloaded.
  • Fixed an issue where user tags would appear as loading spinners if not connected to the internet
  • Fixed an issue where multi-line codeblocks in page drafts would not have their highlight color applied correctly
  • Fixed an issue where there would sometimes be extraneous line breaks in code blocks
