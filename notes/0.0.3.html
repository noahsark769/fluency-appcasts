Fluency 0.0.3

Thank you for beta testing Fluency!

Improvements and Fixes

  • Support for images with outside URL sources. Images which are page attachments coming soon!
  • Fluency's auth system is now more robust, which means your Confluence session should last longer without having to re-login.
  • Tweak styles for monospaced text: it's now easier to see and outlined in gray
  • Fixed: Issue where when filtering the page tree, the search field would lose focus
