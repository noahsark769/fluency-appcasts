Fluency 0.0.16

Thank you for beta testing Fluency!

Improvements and Fixes

  • Fluency now displays task lists and allows you to complete the tasks by clicking the checkboxes
  • When editing drafts, option-left and option-right now respect periods as word breaks
  • Fluency now pre-selects text when you open the search menu so you don't have to delete it to start another search
  • Fluency now displays a button to log in directly in the search results view, if you're logged out
  • Fixed an issue where tagged users would display as 'unknown user'
  • Fix for an issue where local pages would be shown twice in the search results list
  • Fixed an issue where trying to search text with periods in it would cause a crash
  • Fixed an issue where spaces with no pages would show as downloading when they weren't actually downloading
  • Fixed an issue where perioud updates would cause spaces to be deleted if you were logged in with cookie-based authentication
