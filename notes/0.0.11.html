Fluency 0.0.11

Thank you for beta testing Fluency!

Improvements and Fixes

  • Experimental support for viewing tables. Tables may not always look correct, but they will generally render.
  • Space sidebar styles are a little bit prettier
  • Fixed an issue where placeholder strings would sometimes flicker in the page draft editor
  • Fixed an issue where the page draft title would flicker during typing
  • Fixed an issue where nested lists would appear with too many line breaks
  • Fixes an issue where the correct space would not be highlighted after jumping to a page
