Fluency 0.0.17

Thank you for beta testing Fluency!

Improvements and Fixes

  • Fluency now supports rendering horizontal rule (divider) macros
  • Fluency now renders time and date macros correctly
  • Links to favorite and recent pages can now be copied from the Fluency sidebar
  • Fluency now displays Confluence Layouts correctly
  • Fluency now renders status macros correctly
  • Fixed an issue where pasting text into the page draft view would cause the text to look incorrect
  • Fixed an issue where Fluency would sometimes open the wrong page from search
  • Fix for an issue where copying a link from the page tree would not actually copy the link
