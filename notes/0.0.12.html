Fluency 0.0.12

Thank you for beta testing Fluency!

Improvements and Fixes

  • Fluency can now store your Confluence username/password so that you don't have to log in as often.
  • Login page now has a prettier style
  • Command-Q now keeps the Fluency menu item alive (Fluency can be quit completely from the menu item)
  • Fixed an issue where space keys were not visible when adding spaces
  • Fix for an issue where empty spaces would cause an error in the Add Space view
  • Fixes an issue where the page would sometimes not show up after searching from the menu bar icon
  • Fixed an issue where the space list in the sidebar could become misaligned
  • Fixed an issue where deleting a space then searching for pages could cause the app to crash
