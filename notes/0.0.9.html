Fluency 0.0.9

Thank you for beta testing Fluency!

Improvements and Fixes

  • Fluency now supports finding text within a page's content
  • When adding a space, you can now search by space key as well as space title
  • Fixed an issue where copying heading links in pages with special characters in the title would not work
  • Fix for an issue where publishing pages with no content would result in a spinner which spun forever
  • Fixed: The minimum size of the page draft window is now smaller
  • Fixed: Newlines were erroneously displayed under headers in page draft previews
  • Fixed: Tapping on the Fluency dock icon now re-opens the main window
  • Fixed: When pressing enter while editing a list bullet, the bullet is de-indented correctly if there is no content in the bullet
  • Fixed: Pressing tab while editing a list bullet in a draft now indents the line correctly, regardless of line content
  • Fixed: The wrong app name was displayed in the About section of the main menu
