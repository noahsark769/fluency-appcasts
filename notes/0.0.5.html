Fluency 0.0.5

Thank you for beta testing Fluency!

Improvements and Fixes

  • The menu bar item and "Jump To Page" now support searching all of Confluence as well as your local spaces. Non-local results show below local results, in a different section.
  • Pages you visit will be cached so you can see their content without having to re-login. Stale page data is denoted with a banner.
  • The Add Space window has been refactored for a more intuitive user experience
  • The Add Space window automatically selects the search bar after being opened
  • The Add Space window queries personal spaces in addition to global spaces
  • Fix for an issue which caused icons to display incorrectly
  • Fix for an issue where the Add Space window would sometimes not show a loading spinner
  • Fix for an issue where the app would stall if you provided an invalid url when logging in
  • Fix for an issue where table of contents rows would display indirectly if their headings included comments
