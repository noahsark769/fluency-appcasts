Fluency 0.0.4

Thank you for beta testing Fluency!

Improvements and Fixes

  • Support for images, both from outside urls and attachments on the page
  • Fluency has a status bar icon you can use to launch quick search from anywhere
  • There's a settings page where you can configure a global keyboard shortcut to launch the quick search
  • Support for viewing preformatted text and code blocks
  • Fixed: Parts of the page with inline comments now render correctly
  • Fixed: Notes/tips/warnings that have a title now correctly display their content
