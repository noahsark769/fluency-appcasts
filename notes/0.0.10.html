Improvements and Fixes

  • After jumping to a heading from a table of contents, Fluency now highlights the heading

  • Fixed an issue where too many newlines would appear in pages under certain circumstances
  • Fixed an issue where no error would be shown when trying to load favorites spaces in the Add Space window
