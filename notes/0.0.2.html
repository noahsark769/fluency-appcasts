Fluency 0.0.2

Thank you for beta testing Fluency! This version is the first to use automatic updates and release notes.

Improvements

  • Fluency now supports images which are linked by url
  • List items are now shown closer together, which I think is easier to read
  • Click on a heading in any table of contents to jump to that heading
  • Unsupported tags and macros have a new look which makes them easier to read

Fixes

  • Fixed serveral colors which didn't display well in non-dark themes
  • Fixed an issue where while typing in the page tree search field, it would lose focus
  • Fixed a crash when you enter the jump to page window but there are no results
